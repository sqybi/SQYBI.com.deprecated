Something about this site

这个网站由以下服务/技术提供： Linode · Python · Django · MySQL · jQuery · Bootstrap
你也可以在任何一页的页脚看到这些说明。

现在你看到的这个版本的网站在2014年8月14日被第一次commit。在这之前这个网站还曾经有一个Ruby on Rails版本的前身，但是后来因为sqybi觉得RoR太难学而被抛弃了。

虽然commit的时间很早，但是由于 sqybi很懒 某些不可告人的原因，网站实际上是从2015年4月初开始正式进入开发阶段的。

网站会一直添加新功能，更新频率取决于sqybi 有多懒 有多少空闲时间。

好吧其实他真的很懒。
